test if the given ray intersects this object

Parameters:
rthe incoming ray
pointoutput parameter - if there is any intersection, the point of intersection on the surface of the object will be returned through this reference
distoutput parameter - if there is any intersection, the distance from the origin of the ray to the intersection will be returned via this parameter
Returns:
true if any intersection occurs
